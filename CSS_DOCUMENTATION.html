Classes de mise en page

.main-button

Propriete Description
Objectif Bouton d'appel a l'action principal
Apparence Fond blanc, texte noir, police Playfair
Hover Le texte devient dore (--gold)
Position Positionnement absolu (utiliser dans un conteneur relatif)

Utilise dans :

  • home.html (ligne 12) - Bouton "Visitez la boutique"
<a class="main-button" href="#">Visitez la boutique</a>

.main-container

Propriete Description
Objectif Conteneur principal du contenu de la page
Disposition Flexbox, direction colonne, centre
Padding 20px sur tous les cotes

Utilise dans :

  • home.html (ligne 14) - Contenu principal de la page
  • login.html (ligne 9) - Conteneur du formulaire de connexion
  • logout.html (ligne 9) - Conteneur du formulaire de deconnexion
  • signup.html (ligne 9) - Conteneur du formulaire d'inscription
  • forget_password.html (ligne 9) - Conteneur de reinitialisation du mot de passe
<div class="main-container">
    <!-- Contenu de la page ici -->
</div>

.main-tile

Propriete Description
Objectif Conteneur de section de contenu horizontal
Disposition Flexbox, direction ligne, centre
Largeur calc(100% - 160px)
Padding 40px 0 (haut/bas)

Utilise dans :

  • home.html (ligne 19) - Conteneur des tuiles de collection
  • home.html (ligne 57) - Section des nouveaux articles
<div class="main-tile">
    <!-- Contenu de la tuile ici -->
</div>

.text-box

Propriete Description
Objectif Conteneur pour les sections de texte avec titres stylises

Police des paragraphes GB Garamond

Utilise dans :

  • home.html (ligne 15) - Section de texte hero
<div class="text-box">
    <h1>Titre</h1>

</div>

Systeme d'alertes

.alert

Propriete Description
Objectif Conteneur de notification toast
Position Fixe, en bas a gauche (glisse depuis la gauche)
Disposition Flexbox avec gap
Animation Glissement via transition left (1s)
Z-index var(--alert-zindex) (300)

Utilise dans :

  • layout.html (ligne 28) - Notifications de messages flash
  • alerts.js - Animation et comportement de fermeture
<div class="alert alert-success" role="alert">
    <span>Message</span>
    <button>x</button>
</div>

.alert-warning

Propriete Description
Objectif Variante de notification d'avertissement
Fond Dore (--gold)
A combiner avec : .alert
<div class="alert-warning alert">Message d'avertissement</div>

.alert-danger

Propriete Description
Objectif Variante de notification d'erreur/danger
Fond Rouge (rgb(189, 51, 51))
A combiner avec : .alert
<div class="alert-danger alert">Message d'erreur</div>

.alert-success

Propriete Description
Objectif Variante de notification de succes
Fond Vert (rgb(71, 176, 48))
A combiner avec : .alert
<div class="alert-success alert">Message de succes</div>

styles-utilities.css

Classes utilitaires avec !important pour des surcharges rapides.

.no-margin

Propriete Description
Objectif Supprimer toutes les marges
Valeur margin: 0 !important 
<p class="no-margin">Texte sans marge</p>

.no-padding

Propriete Description
Objectif Supprimer tout le padding
Valeur padding: 0 !important

Utilise dans :

  • _nav.html (ligne 20) - Bouton de recherche
<button class="button-search-bar no-padding">search</button>

.red-text

Propriete Description
Objectif Rendre le texte rouge (pour les indicateurs de champs obligatoires)
Valeur color: red !important

Utilise dans :

  • login.html (ligne 14) - Asterisque de champ obligatoire
  • signup.html (ligne 14) - Asterisques de champs obligatoires
  • forget_password.html (ligne 14) - Asterisque de champ obligatoire
<label>Nom d'utilisateur <span class="red-text">*</span></label>

.no-link-style

Propriete Description
Objectif Supprimer le style par defaut des liens
Couleur Blanc
Decoration du texte Aucune
Heritage S'applique a tous les elements enfants (*)

Utilise dans :

  • home.html (ligne 20) - Liens des conteneurs de collection
  • home.html (ligne 36) - Liens des conteneurs de collection
<a class="collection-container no-link-style" href="/">
    <span>Texte de lien sans soulignement</span>
</a>
